## Uninstall ehdd:

Uninstallation of **ehdd** depends on which method you used for installation.

#### Uninstall when you used [Method-1](./INSTALL.md#method-1)

1. Open up a terminal in the directory which contains `ehdd`.
2. Run this command in the terminal-
   `$ ./uninstall.sh`

This would uninstall `ehdd` from your device.

#### Uninstall when you used [Method-2](./INSTALL.md#method-2)

1. Open up a terminal in the directory which contains `ehdd`'s source code.
2. Run this command in the terminal- `$ make uninstall`

This would uninstall `ehdd` from your device.
